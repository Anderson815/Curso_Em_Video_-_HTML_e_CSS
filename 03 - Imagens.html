Tag IMG

A Tag IMG ela representa que será utilizada imagem, pense que img vem de imagem, dentro dela podemos colocar alguns atributos, vamos falar de 3 especifícos, SRC, ALT e TITLE

  • SRC

    • Esse atributo é responsável por indicar aonde está a imagem, qual é a origem, o lugar onde se localiza, isso é colocado no valor de SRC. O ideal é que tudo do seu site esteja na pasta principal ou em subpastas da pasta principal.

    Se ele estiver na mesma pasta basta escrever o nome do arquivo com o formato (alguns exemplos seria o .jpg ou .png)e pronto, caso esteja em uma subpasta da pasta principal escreva o nome da pasta/arquivo.formato ou seja indique o caminho a partir da pasta principal, caso precise subir um nível isto é sair de uma pasta, ou voltar a pasta anterior use dois pontos ../arquivo.formato, assim você ira voltar uma pasta da qual está a página que está sendo usada

  • ALT

    • Pensa em ALT como um atributo ALTernativo, no valor dele vôcê ira escrever um texto para que caso a imagem não seja carregada, seja por que o valor do SRC esteja errado ou porque o arquivo é muito grande e ainda está carregando, ira aparecer esse texto como uma alternativa para a ausência da imagem

  • TITLE

    • Esse atributo mostro o título da imagem, isto é, quando o usuário passar o mouse por cima da imagem irá aparecer uma janelinha mostrando o título e isso não serve só para imagens, mas também para outros tipos de midia

Tag FIGURE

Essa tag representa conteúdo independente, frequentemente com uma legenda(<"FIGCAPTION">), e é normalmente referido como uma única unidade. Enquanto ela está relacionada com o fluxo principal, sua posição é independente do fluxo principal. Normalmente, isso é uma imagem, uma ilustração, um diagrama, um trecho de código ou um esquema que é referenciado no texto principal, mas que pode ser movido para outra página ou para um apêndice, sem afetar o fluxo principal

Cuidado com a tentação de aplicar em todas suas imagens a tag figure, pois isso não é correto. Para utilizar melhor esse elemento você deve pensar em como a imagem é utilizada na sua página. Se a imagem faz parte do conteúdo da página e for movida para outro local sem perder seu significado, é sinal que ela pode ficar dentro do novo elemento figure. Caso a imagem seja um íncone de navegação, um logotipo, imagens que devem estar num local específico não devem estar marcadas com o elemento figure.

Tag FIGCAPTION

Útilizado para marcar legendas para conteúdos inseridos com uso do elemento FIGURE. Sempre deve vir entre um elemento FIGURE, ou seja, deve constar como elemento-filho do elemento FIGURE.
