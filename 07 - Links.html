Links

Introdução

Você já se deparou com a situação em que você está em um site e no conteudo deste site ele te leva para outra página do próprio site ou para sites diferentes? Então esses são os links que permite essa ligação de uma página para a outra.

Criação

Para fazer liks é necessário utilizar a tag A é nela que iremos fazer os links, só que ela sozinha não tem efeito por isso você irá precisar do parametro HREF você já utilizou esse parametro na tag LINK para o arquivo CSS, essa tag indica o caminho, o local, como está dentro da tag A e não da tag LINK, ela irá indicar para aonde aquele link está apontando, nessa você pode colocar o endereço de outra página do seu site ou de outros sites.

Porém, se você só colocar esse parametro ele irá mudar a sua aba para o endereço da página do link, isso é se por acaso o seu site tiver um link com outro site, então, o seu site irá parar de ser exibido e passará a ser o outro site, um exemplo foi o link posto nessa página no primeiro paragráfo. Mas caso você queira que o link abra uma nova página em outra aba você terá que usar o parametro TARGET com o valor _blank.
