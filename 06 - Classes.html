Classes das tag para CSS

Paragráfo de classe 1

Toda vez que você faz a formatação, isto é como você quer deixar o conteúdo, você coloca uma tag e a formatação desejada para ela, porém essa formatação vai servir para todo contéudo que utilizam aqueles tipos de Tag, então exemplificando esta situação, vamos supor que você fez dois paragráfos e quer formatar eles de forma diferentes, mas a forma que você faz lá no CSS é p{font-family: Arial, font-size:14 e etc}, porém essa formatação vai servir para todo o contéudo que usar a tag P, mesmo que você queira formatalos de formas diferentes.

Paragráfo de classd 2

Para resolver essa situação você pode, no HTML dentro da tag desejada, criar uma classe para aquela tag, no exemplo citado são dois paragráfos que serão formatados de forma diferente, então no HTML você pode fazer assim <'p' class="p1">.....<'/p'> e no outro paragráfo algo como <'p' class="p2">.....<'/p'>. No CSS para poder configurar de forma diferente você irá fazer assim tag.classe_da_tag, exemplo p.p1(tag paragráfo e classe p1), assim somente a tag P que possuir a classe p1 terá aquela formatação
