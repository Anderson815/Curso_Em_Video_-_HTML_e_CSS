Arquivo CSS anexado ao arquivo HTML

Uma coisa interessante é que podemos fazer um arquivo CSS que contém todos as configurações da página de forma separada do HTML, isto é, um CSS que não é local, mas sim global em que se eu colocar o endereço desse arquivo automaticamente a minhas página irá recebera aquelas configuração que foi feita naquele arquivo.

De forma rezumida, podemos escrever em um arquivo CSS um único estilo CSS que pode ser usado para todas as páginas que eu quiser.

Para fazer isso você terá que colocar dentro da Tag Head uma Tag Link e colocar três valores para três atributos: REL, TYPE e HREF:

  • REL

    • REL é um atributo que indica o tipo de RELação tem um documento externo, que será utilizado pelo parametro HREF, que está vinculado com o documento atual. No caso do CSS terá um relacionamento stylesheet (folha de estilo), ou seja, dara a aparência desejada.

  • TYPE

    • Ele indica qual é o TIPO do documento que está sendo vinculado com o documento atual. Esse parametro pode possuir os valores do tipo MIME: text/html, text/css e assim por diante.

  • HREF

    • Ele indica aonde que está o documento que será vinculado ao documento atual, pensa em href como REFerência.

Irá fical algo do tipo <&link rel="stylesheet" type="text/css" href="CSS/04.css"/>
