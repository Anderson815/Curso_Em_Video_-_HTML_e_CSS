Introdução

É muito comum sites utilizarem listas para especificar alguma coisa, por exemplo, listar os melhores times do ano na ordem do melhor para o pior , listar aréas de estudos, listar pesoas que compraram em super mercado, enfim listas é algo muito importante e muito utilizado em sites e por isso é importante você saber faze-lo.

Existe dois tipos de listas mais usadas no HTML, a ORDENADA e a NÃO ORDENADA, cada um com uma semântica especifica

Lista Ordenada

As listas ordenas são as listas que usam numeros , normalmente usadas em rankings, em que um item teve mais destaque que o outro em base em algum critério.

Por exemplo: os 5 melhores jogos do super nintendo:

  1. Zelda the link to the past
  2. Super Mario World
  3. Bomberman 3
  4. Mortal Kombat 3
  5. Top Gear

Para fazer uma lista ordenada você irá usar a tag OL, pensa que o O é de ordenada e o L é de lista se ler OL invertido fica lista ordenada, e para fazer os itens você irá usar a tag LI, pensa que L é de lista e o I de item se ler LI invertido fica item da lista.

Tipos de marcadores da lista ordenada

Nessa lista existe 3 tipos de marcadores de ordem pelos seus valores, ela é referenciada atráves do parametro TYPE e pode possuir os seguintes valores: A, I e 1. Cada um usa marcadores diferentes

Tipo A

Esse marcador indica que a ordem vai ser alfabetica, ou seja, o primeiro item vai estar marcado com o A, o segundo vai estar marcado com o B e assim por diante

  1. primeiro item
  2. segundo item
  3. terceiro item
  4. quarto item
  5. quinto item

Tipo I

Esse marcador utiliza os numeros romanos para poder marcar os itens, ou seja, o primeiro vai estar marcado com o I, o segundo vai estar marcado com o II e assim por diante

  1. primeiro item
  2. segundo item
  3. terceiro item
  4. quarto item
  5. quinto item

Tipo 1

Esse marcador indica a ordem númerica que as pessoas utilizam no dia a dia, ou seja, o primeiro item vai estar marcado com o 1, o sengo item com o marcador 2 e assim por diante

  1. primeiro item
  2. segundo item
  3. terceiro item
  4. quarto item
  5. quinto item

Start

O START é outro parametro da lista ordenada nela você consegue indicar com qual numero, numero romano ou letra que ira começar a lista, mas só pode colocar valores numericos para indica a posição que deseja começar.

Se for numerico e o start tiver o valor 4 então a lista começara no 4, se for romano a lista começara no IV, se for alfabético começara no D

  1. primeiro item
  2. segundo item
  3. terceiro item
  1. primeiro item
  2. segundo item
  3. terceiro item
  1. primeiro item
  2. segundo item
  3. terceiro item

Lista Não ordenada

As listas não ordenadas são listas que eu quero apenas citar os itens, em que nenhum tem o peso maior que o outro como na lista ordenada, é como se todos fosem iguais, por isso os marcadores não indicam uma ordem

Por exemplo: O que tem em o gabinete do PC

  • CPU
  • Memória RAM
  • Memória ROM
  • HD
  • Barramento

Para fazer uma lista NÃO ORDENADA você ira utilizar a tag UL, e dentro dela a tag LI para fazer os itens da listas igual na lista ordenada

Tipos de marcadores

Os marcadores dessa lista são simbolos para não indicar preferencia ou peso de um item ele possui 3 tipos de marcadores, sendo referenciada com a tag TYPE e com os valores: Circle, Square e Disc

Tipo Cicle

Esse marcador indica um circulo com o fundo não preencido, ou seja somente a borda

  • primeiro item
  • segundo item
  • terceiro item
  • quarto item
  • quinto item

Tipo Square

Esse marcador é um quadrado todo preencido

  • primeiro item
  • segundo item
  • terceiro item
  • quarto item
  • quinto item

Tipo Disc

Esse marcador indica um circulo todo preenchido

  • primeiro item
  • segundo item
  • terceiro item
  • quarto item
  • quinto item

Observação, nas listas não ordenas não existe o parametro start já que todos os itens são marcados com o mesmo marcador
